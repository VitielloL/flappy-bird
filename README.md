# 🐦 Flappy Cacau

Uma implementação moderna do clássico jogo Flappy Bird, desenvolvida com **HTML5**, **CSS3** e **JavaScript puro**. Teste suas habilidades em um jogo desafiador e viciante!

![Flappy Cacau](assets/print.bmp)

## 🎮 Sobre o Jogo

Flappy Cacau é uma interpretação do famoso jogo casual Flappy Bird. Controle o pássaro (nosso querido cacau), evite as barreiras e acumule o máximo de pontos possível. Com mecânica simples mas desafiadora, é perfeito para colocar seus reflexos à prova!

## ✨ Características

- 🎯 **Gameplay Clássico**: Mecânica simples e intuitiva
- 🎨 **Design Limpo**: Interface visual agradável e responsiva
- ⚡ **Performance Otimizada**: Desenvolvido com JavaScript vanilla, sem dependências
- 📊 **Sistema de Pontuação**: Acompanhe seu melhor score
- 🔄 **Jogabilidade Infinita**: Desafio cada vez maior conforme você progride
- 📱 **Browser Compatible**: Funciona em qualquer navegador moderno

## 🕹️ Como Jogar

1. **Iniciar o Jogo**: Abra o arquivo `index.html` no seu navegador
2. **Controlar**: Pressione qualquer tecla ou clique para fazer o pássaro voar para cima
3. **Objetivo**: Desvie das barreiras e ganhe pontos passando entre elas
4. **Game Over**: Colida com uma barreira ou com as laterais para terminar
5. **Reiniciar**: Pressione `F5` para começar um novo jogo

## 🚀 Como Rodar Localmente

### Pré-requisitos
- Um navegador web moderno (Chrome, Firefox, Edge, Safari)

### Instalação
1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/flappy-bird.git
cd flappy-bird
```

2. Abra o arquivo `index.html`:
   - Dê duplo clique em `index.html`, ou
   - Clique com botão direito e selecione "Abrir com navegador", ou
   - Use um servidor local (recomendado):

```bash
# Com Python 3
python -m http.server 8000

# Ou com Python 2
python -m SimpleHTTPServer 8000

# Ou com Node.js (http-server)
npx http-server
```

3. Acesse no navegador:
```
http://localhost:8000
```
